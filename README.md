# 🎯 Real-Time Object Detection

A real-time object detection system built using Python, OpenCV, and YOLOv8. The application detects multiple objects from a live webcam feed and displays bounding boxes with class labels and confidence scores.

## Features

- Real-time object detection
- YOLOv8 deep learning model
- Detects 80+ object classes
- Bounding boxes with confidence scores
- Live webcam support
- Press `Q` to exit

## Technologies Used

- Python
- OpenCV
- YOLOv8 (Ultralytics)
- NumPy

## Installation

```bash
git clone https://github.com/abhinavashish1710/real-time-object-detection.git
```

```bash
pip install -r requirements.txt
```

```bash
python object_detection.py
```
